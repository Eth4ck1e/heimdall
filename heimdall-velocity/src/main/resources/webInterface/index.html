About

This plugin is designed to provide ticket and applications management functionality for Discord while also synchronizing whitelists between Paper/Spigot servers and Velocity proxy servers. It utilizes a MySQL database for data storage and offers additional features for seamless integration with both Paper/Spigot and Velocity servers.

For more information and detailed documentation, please refer to the official project documentation or the plugin's GitHub repository.
